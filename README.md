# 🚦 Traffic Vision

> Real-time vehicle & pedestrian detection system powered by YOLOv8

## 📌 Problem Statement
Urban traffic monitoring is manual & inefficient. 
This system automates vehicle detection in real-time.

## 🎯 Features
- Real-time vehicle detection (cars, trucks, bikes, buses)
- Pedestrian detection & counting
- FPS & confidence score display
- Webcam + video file support
- Live analytics dashboard

## 🛠️ Tech Stack
- YOLOv8 (Ultralytics)
- OpenCV
- PyTorch
- Streamlit
